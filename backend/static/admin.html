API 接口文档

供输入法 App 调用，配置 Tailscale 后手机可远程访问

配置 Tailscale 后替换为 http://<你的Tailscale IP>:8000
📡 接口说明
• GET /api/scripts — 话术列表（支持 category/keyword/page 参数）
• POST /api/scripts — 创建话术
• GET /api/scripts/{id} — 获取话术详情（自动 +1 使用次数）
• PUT /api/scripts/{id} — 更新话术
• PATCH /api/scripts/{id}/favorite — 切换收藏
• DELETE /api/scripts/{id} — 删除话术
• POST /api/ai/stream — AI 流式处理（SSE）
• GET /api/ai/modes — 获取 AI 模式列表

📱 手机端连接
1. 电脑安装 Tailscale
2. 手机也装 Tailscale，登录同一账号
3. 手机输入法设置里填入 Tailscale IP + :8000

设置

配置 DeepSeek API Key（AI 润色需要）

⚠️ 重要：API Key 配置
DeepSeek API Key 是 AI 润色功能的必要配置。
申请地址：platform.deepseek.com
新用户注册送 ¥10-15，可处理数万条消息。
# 在后端服务器上运行一次即可（永久生效）
set DEEPSEEK_API_KEY=sk-your-key-here

📱 如何让手机使用话术库

1. 电脑上运行 python -m uvicorn app.main:app --reload

2. 手机安装 Tailscale，登录同一账号

3. 在输入法 App 设置里填入 http://100.x.x.x:8000（Tailscale IP）

⚠️ 注意：电脑关机时后端不可用，手机也就用不了 AI 功能
